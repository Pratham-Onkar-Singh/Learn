FLEX-BOX

Flexbox (Flexible Box Layout) is a CSS layout model that allows you to design complex layouts with flexible and responsive arrangements by distributing space within a container, aligning items, and handling their resizing.

POSITION PROPERTY

The position property in CSS specifies how an element is positioned within its containing element, determining its position relative to the normal document flow, the viewport, or its nearest positioned ancestor.

GRID

A grid in CSS is a layout system that allows you to create complex, two-dimensional web layouts using rows and columns, enabling precise control over the placement and alignment of elements within a container.
